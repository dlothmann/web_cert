# web_cert
Simple implementation for generating SSL wildcard certificates with IONOS API.

Based on https://www.devlix.de/lets-encrypt-wildcard-zertifikate-mit-ionos-dns-api-erzeugen/
---
INSTALL AS ROOT

clone to /opt/
chmod +x install.sh
./install.sh

## Requirements
- python 3
- docker
- curl
- powershell
- tar
